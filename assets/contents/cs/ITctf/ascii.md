# 아스키 코드(ASCII Code)

## 숫자
- `'0'` ~ `'9'`: 48 ~ 57

## 대문자
- `'A'` ~ `'Z'`: 65 ~ 90

## 소문자
- `'a'` ~ `'z'`: 97 ~ 122

## 변환 공식
```
대문자 → 소문자: +32
소문자 → 대문자: -32

'A' + 32 = 'a' (65 + 32 = 97)
'a' - 32 = 'A' (97 - 32 = 65)
```

## 주요 특수문자
- 공백(Space): 32
- `!`: 33
- `+`: 43
- `-`: 45
- `.`: 46
- `/`: 47

## `\0` (Null Character)

**아스키 코드: 0**

## 의미
- **문자열 종료 표시자(String Terminator)**
- C/C++에서 문자열의 끝을 나타냄

## 예시
```c
char str[] = "Hello";
// 실제 메모리: ['H','e','l','l','o','\0']
```

## 특징
- 출력되지 않는 문자
- `strlen()` 계산 시 제외
- 문자열 함수들이 끝을 인식하는 기준
